# Sentinel: Intelligent Access Control System 
A public, self-testable security gateway with real and simulated attack traffic

---

## Overview
Sentinel is an intelligent access control system designed to detect, analyze, and respond to both real and simulated attack traffic. It provides a comprehensive security gateway with self-testing capabilities to validate security measures.

## Features
- Real-time threat detection and analysis
- Simulated attack traffic testing
- Intelligent access control mechanisms
- Self-testable security framework
- Interactive security dashboard

## Dependencies Used
- react-router-dom
- axios
- bootstrap
- lucide-react
- react-hot-toast

## Support Repository
Backend repository: [Sentinel](https://github.com/Thecoderkushagra/Sentinel)

## Getting Started
Ensure all dependencies are installed and configured. The application integrates with the backend API for comprehensive security monitoring and threat management.

## How It Works
The system utilizes a frontend-backend architecture where the frontend provides an interactive interface for monitoring and controlling access policies, while the backend handles threat detection and security logic.
